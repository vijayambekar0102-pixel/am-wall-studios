# am-wall-studios

## Project Description
This project is designed to showcase the works of Am Wall Studios, a creative agency focused on delivering innovative design solutions.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/am-wall-studios.git
   cd am-wall-studios
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Available Scripts
- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm run start`: Starts the production server.

## Deployment Guidelines for Vercel
1. Sign in to your Vercel account.
2. Import your GitHub repository.
3. Follow the prompts to set up your deployment environment. 
4. Vercel will automatically detect the framework and set the necessary configuration.
5. Click "Deploy" and your application will be live!
